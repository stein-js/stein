# Stein CLI

## Usage

```bash
npx @steinjs/cli
pnpx @steinjs/cli
bunx @steinjs/cli
```

## Commands

```bash
stein create project-name # Create a new Stein project in "project-name" directory
stein dev # Start the development server

# todo:
stein add plugin-name # Add plugin-name to the current project.
stein build
```
